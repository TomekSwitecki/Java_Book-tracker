# Java Book Tracker

A full-stack book tracking application built to learn modern Java backend development.

## Tech Stack

* Frontend: React, TypeScript, Vite
* Backend: Java, Spring Boot
* Database: PostgreSQL
* ORM: Spring Data JPA / Hibernate
* Hosting: Render (Frontend), Railway (Backend & Database)

## Features

* View all books
* Add new books
* Delete books
* Store data in PostgreSQL
* RESTful API built with Spring Boot

## Purpose

This project was originally built with an Express.js and SQLite backend and later migrated to Java Spring Boot with PostgreSQL to gain hands-on experience with enterprise Java development, REST APIs, JPA/Hibernate, and cloud deployment.
